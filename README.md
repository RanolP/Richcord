# Richcord

Awesome rich presence manager

## Features

 * Custom Game
   * Custom image (You can change it dynamically)
   * Custom title
   * Custom Details
   * Custom states
   * Custom time checking modes
     * Elapsed mode
     * Left mode
     * No time checking
